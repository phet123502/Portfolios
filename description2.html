About Tour Packages

Project Description

This website is designed to display various tourist destinations and their packages in a simple and visually appealing manner. It uses Bootstrap components to create a responsive layout, making it easy to view on both desktop and mobile devices. The website allows users to explore destinations, check prices, and book their dream vacations.

Technologies Used

  • HTML5 – Provides the structure and layout of the website.
  • CSS3 – Adds styles, colors, and spacing to enhance the look.
  • Bootstrap 5 – Used for responsive design, cards, and navigation bar.
