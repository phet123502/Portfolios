Startup Project Showcase

A brief overview of the technologies and design used in the project.

Technologies Used

  • HTML5
  • CSS3 (Custom stylesheet: p77.css)
  • Bootstrap 5.3.1 (via CDN)
  • Google Fonts – Montserrat

Project Description

This project represents a simple yet elegant startup landing page named “Startup”. It uses modern web technologies like Bootstrap 5 for layout and responsive design, along with Google Fonts for clean typography. The purpose of the page is to highlight a startup’s main vision or event, presented through a minimalistic structure and styled using a separate custom CSS file.
